About Me

Hi, my name is Joseph Chan and I'm a freelance front end developer based in Edmonton.

I turn your ideas to life on the browser. My skills include HTML5, CSS3, and Bootstrap 3. Continuous improvement is my motto, so I'm currently adding JavaScript and jQuery to my skillset.

My interests include responsive web design, web design systems (specifically Atomic Design), and user experience.

I'm fascinated by how web design is adapting to growing device diversity, i.e. mobile devices, and how web technologies like CSS3 and Bootstrap are responding to this change.

I'm currently offering pro bono work to grow my portfolio.

Contact me today and let's start a conversation on how we can build your web ideas.
